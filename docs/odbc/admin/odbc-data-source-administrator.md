# <a name="odbc-data-source-administrator"></a>ODBC-Datenquellen-Administrator
Der Microsoft® ODBC-Datenquellenadministrator verwaltet Datenbanktreiber und Datenquellen. Diese Anwendung befindet sich in der Windows-Systemsteuerung unter **Verwaltungstools**. Ab Windows 8 heißt das Symbol **ODBC Data Sources**, und auf 64-Bit-Betriebssystemen gibt es eine 32-Bit- und 64-Bit-Version.  
  
 Informationen zu detaillierten ODBC-Administratorprozeduren erhalten Sie im Dialogfeld [ODBC Data Source Administrator,](https://msdn.microsoft.com/eea94d94-f53b-4289-ae75-9ccccde15333) und klicken Sie auf **Hilfe**.  
  
 Sie können PowerShell-Befehle verwenden, um Treiber und Datenquellen zu verwalten. Weitere Informationen zu diesen PowerShell-Befehlen finden Sie unter [PowerShell-Befehle für Windows Data Access Components](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx).
