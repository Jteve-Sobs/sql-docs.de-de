# <a name="viewing-drivers"></a>Anzeigen von Treibern
Zum Konfigurieren von Datenquellen müssen Sie mindestens einen Treiber auf Ihrem System installiert haben. Auf der Registerkarte **Treiber** im Dialogfeld **ODBC-Datenquellenadministrator** werden alle auf Ihrem Computer installierten Treiber aufgeführt, einschließlich Name, Version, Unternehmen, Dateiname und Dateierstellungsdatum der einzelnen Treiber. Sie müssen das Setupprogramm des Treibers verwenden, um einen Treiber zu ihrem System hinzuzufügen oder zu löschen. Weitere Informationen zum Ändern von Treibern finden Sie unter [Verwalten von Datenquellen](../../odbc/admin/managing-data-sources.md).
