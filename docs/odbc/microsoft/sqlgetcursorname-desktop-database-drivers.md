# <a name="sqlgetcursorname-desktop-database-drivers"></a>SQLGetCursorName (Desktop-Datenbanktreiber)
**SQLGetCursorName** wird unterstützt, aber nur verwendet werden, wenn die Cursorbibliothek verwendet wird, da positionierte Operationen in der Treiber nicht unterstützt.
