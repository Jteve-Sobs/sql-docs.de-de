# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Erstellen und Löschen von Sichten und Prozeduren mit ExtendedAnsiSQL
Wenn das extendedansisql-Flag aktiviert ist, unterstützt Jet 4,0 die Möglichkeit, Sichten und Prozeduren mithilfe von SQL zu erstellen und zu löschen. Diese Sichten und Prozeduren werden nicht durch den Zugriff sichtbar.
