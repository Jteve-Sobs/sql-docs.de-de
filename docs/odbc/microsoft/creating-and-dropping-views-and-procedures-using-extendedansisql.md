# <a name="creating-and-dropping-views-and-procedures-using-extendedansisql"></a>Erstellen und Löschen von Sichten und Prozeduren mit ExtendedAnsiSQL
Wenn das ExtendedAnsiSQL-Flag aktiviert ist, unterstützt Jet 4.0 die Möglichkeit, Ansichten und Prozeduren über SQL zu erstellen und zu löschen. Diese Ansichten und Prozeduren sind über Access nicht sichtbar.
