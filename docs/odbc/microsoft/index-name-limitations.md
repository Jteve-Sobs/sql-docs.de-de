# <a name="index-name-limitations"></a>Einschränkungen für Indexnamen
Wenn der Paradox-Treiber verwendet wird, muss ein primärer Index denselben Namen wie die Tabelle haben, für die er definiert ist. Andere eindeutige oder nicht eindeutige Indizes müssen denselben Namen wie die Tabelle haben, für die sie definiert sind.
