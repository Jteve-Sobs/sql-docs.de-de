# <a name="multiple-hstmts-paradox-driver"></a>Mehrere hstmts (Paradox-Treiber)
Wenn die ODBC-Paradox-Treiber verwendet wird, wenn Sie mehr als eine verwenden möchten *Befehls beschäftigt* um Abfragen für eine Tabelle auszuführen, muss die Tabelle einen eindeutigen Index (Paradox-Primärschlüssel) haben.
