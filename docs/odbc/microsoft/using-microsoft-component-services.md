# <a name="using-microsoft-component-services"></a>Verwenden von Microsoft Component Services
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Version von Windows entfernt. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den von Oracle bereitgestellten ODBC-Treiber.  
  
 Sie können eine Oracle-Datenbank für die Zusammenarbeit mit Transaktions Komponenten Diensten (oder MTS, wenn Sie Windows NT verwenden) unter Microsoft Windows NT/Windows 2000 und Microsoft Windows 95/98 aktivieren. Damit eine Oracle-Datenbank mit Komponenten Diensten funktioniert, die Transaktionen unterstützen, sollten Systemadministratoren eine Ansicht mit dem Namen V $ xatrane $ erstellen. Zum Erstellen dieses Skripts müssen Sie ein von Oracle bereitgestelltes Skript ausführen. Weitere Informationen finden Sie in der Hilfe zu den Komponenten Diensten oder der Oracle-Dokumentation.
