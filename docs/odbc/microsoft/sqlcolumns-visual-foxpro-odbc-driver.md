# <a name="sqlcolumns-visual-foxpro-odbc-driver"></a>SQLColumns (Visual FoxPro-ODBC-Treiber)
> [!NOTE]  
>  Dieses Thema enthält Visual FoxPro-ODBC-Treiber-spezifische Informationen. Allgemeine Informationen zu dieser Funktion finden Sie unter den entsprechenden Themen unter [ODBC-API-Referenz](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Support: Vollständig  
  
 ODBC-API-Übereinstimmung: Ebene 1  
  
 Für eine Tabelle erstellt ein Resultset, das der Liste der Spalten für die angegebene Tabelle oder Tabellen.  
  
 Weitere Informationen finden Sie unter [SQLColumns](../../odbc/reference/syntax/sqlcolumns-function.md) in die *ODBC Programmer's Reference*.
