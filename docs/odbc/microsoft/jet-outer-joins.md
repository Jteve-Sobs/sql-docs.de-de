# <a name="jet-outer-joins"></a>Jet: Äußere Verknüpfungen
Eine SELECT-Anweisung kann es sich um eine Liste der OUTER JOIN-Klauseln enthalten. Geschachtelte ÄUßERE JOINS werden unterstützt. Weitere Informationen zu ÄUßEREN JOINS, finden Sie unter den *Microsoft Jet-Datenbank-Engine Handbuch für Programmierer*.
