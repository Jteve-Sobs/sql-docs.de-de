# <a name="schema-text-file-driver"></a>Schema (Textdateitreiber)
Das Schema enthält Informationen zu jeder Tabelle (Textdatei) in einer Datenquelle, einschließlich des Tabellenformats, der Anzahl der zu scannenden Zeilen zum Bestimmen von Spaltentypen, der Frage, ob die erste Zeile der Tabelle Spaltennamen enthält, ob die Quelldatei mithilfe einer OEM- oder ANSI-Codepage geschrieben wird, sowie den Namen, den Datentyp und die Breite jeder Spalte.
