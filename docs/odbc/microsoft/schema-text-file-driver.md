# <a name="schema-text-file-driver"></a>Schema (Textdateitreiber)
Das Schema enthält Informationen zu jeder Tabelle (Textdatei) in einer Datenquelle, einschließlich des Tabellen Formats, der Anzahl der zu scannenden Zeilen, um die Spaltentypen zu ermitteln, ob die erste Zeile der Tabelle Spaltennamen enthält, ob die Quelldatei mithilfe eines OEM-oder ANSI-Codepage und der Name, Datentyp und die Breite jeder Spalte.
