# <a name="native-error"></a>Native Fehler
Positive systemeigene Fehlercodes werden von den Desktopdatenbanktreibern generiert. Negative Fehlercodes werden von Microsoft Jet generiert. Weitere Informationen finden Sie im *Microsoft Jet Database Engine Programmer es Guide*.
