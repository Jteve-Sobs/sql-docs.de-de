# <a name="bookmark-support-visual-foxpro-odbc-driver"></a>Lesezeichenunterstützung (Visual FoxPro-ODBC-Treiber)
Der Visual FoxPro-ODBC-Treiber unterstützt die einfache Lesezeichen. Beim Aufruf [SQLGetInfo](../../odbc/microsoft/sqlgetinfo-visual-foxpro-odbc-driver.md) mit der SQL_BOOKMARK_PERSISTENCE *Informationsart*, SQL_BP_SCROLL zurückgegeben wird.  
  
 Weitere Informationen über Lesezeichen finden Sie unter [Lesezeichen (ODBC)](../../odbc/reference/develop-app/bookmarks-odbc.md).
