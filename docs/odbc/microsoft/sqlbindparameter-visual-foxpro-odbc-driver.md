# <a name="sqlbindparameter-visual-foxpro-odbc-driver"></a>SQLBindParameter (Visual FoxPro-ODBC-Treiber)
> [!NOTE]  
>  Dieses Thema enthält Visual FoxPro-ODBC-Treiber-spezifische Informationen. Allgemeine Informationen zu dieser Funktion finden Sie unter den entsprechenden Themen unter [ODBC-API-Referenz](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Support: Vollständig  
  
 ODBC-API-Übereinstimmung: Ebene 1  
  
 Ordnet einen Puffer mit einer parametermarkierung in einer SQL­Anweisung. Der Visual FoxPro-ODBC-Treiber unterstützt Eingabeparameter gemäß der *fParamType* Argument.  
  
 Weitere Informationen finden Sie unter [SQLBindParameter](../../odbc/reference/syntax/sqlbindparameter-function.md) in die *ODBC Programmer's Reference*.
