# <a name="creating-mailing-labels-in-microsoft-word-using-visual-foxpro-data"></a>Erstellen von Adressetiketten in Microsoft Word mithilfe von Visual FoxPro-Daten
Sie können Visual FoxPro-Daten in einem Dokument mit Microsoft Word für Windows 95 oder Windows 98 verwenden. Beispielsweise können Sie Mailing-Bezeichnungen aus den Kundeninformationen erstellen, die in einer Visual FoxPro-Tabelle gespeichert sind.  
  
### <a name="to-create-mailing-labels"></a>So erstellen Sie Mailing-Bezeichnungen  
  
1.  Erstellen Sie in Microsoft Word ein neues leeres Dokument.  
  
2.  Wählen Sie im Menü Extras die Option e-Mail-Merge aus.  
  
3.  Klicken Sie im Hilfsprogramm für die Nachrichten Zusammenführung auf erstellen, und wählen Sie dann Post Schriften.  
  
4.  Wählen Sie unter Hauptdokument die Option aktives Fenster aus.  
  
5.  Wählen Sie unter Datenquelle die Option Daten beziehen und dann Datenquelle öffnen aus.  
  
6.  Wählen Sie im Dialogfeld Datenquelle öffnen die Option MS Query aus.  
  
7.  Wählen Sie im Dialogfeld Datenquelle auswählen eine Visual FoxPro-Datenquelle aus, und klicken Sie dann auf verwenden.  
  
8.  Wenn die Datenbank, auf die von der Datenquelle zugegriffen wird, Tabellen enthält, wählen Sie im Dialogfeld Tabellen hinzufügen eine Tabelle aus. Microsoft Query zeigt die hinzugefügte Tabelle in der oberen Hälfte des Abfrage-Designers an.  
  
9. Wählen Sie Felder für die Abfrage aus, indem Sie Sie aus der Tabelle in die untere Hälfte des Designers ziehen.  
  
10. Wählen Sie im Menü Datei die Option Daten an Microsoft Word zurückgeben aus. Die Microsoft-Abfrage wird geschlossen, und die von Ihnen ausgewählten Daten sind für die Verwendung in Ihrem e-Mail-Zusammenschluss Dokument verfügbar  
  
11. Klicken Sie unter Hauptdokument auf Setup.  
  
12. Wählen Sie im Dialogfeld Bezeichnungs Optionen die gewünschten Drucker-und Bezeichnungs Informationen aus, und klicken Sie dann auf OK.  
  
13. Wählen Sie im Dialogfeld Bezeichnungen erstellen die Felder aus, die Sie auf den Mailing-Bezeichnungen drucken möchten, und klicken Sie dann auf OK.  
  
14. Klicken Sie im Hilfsprogramm für die Nachrichten Zusammenführung unter dem Dokument Daten mit dem Dokument zusammenführen auf Zusammenführen.  
  
15. Wählen Sie im Dialogfeld Zusammenführen die gewünschten Optionen aus, und klicken Sie dann auf Zusammenführen.
