# <a name="product-support"></a>Produktsupport
Produktsupport für ODBC wird von Microsoft Product Support Services bereitgestellt. Da viele Microsoft-Produkte verwenden von ODBC als Kernkomponente und ODBC-Treiber auch verteilen, berücksichtigt Microsoft Product Support Services ODBC Teil jedes Produkts sein, die ODBC als eine Komponente umfasst. Unterstützung für ODBC in Übereinstimmung mit dem supportvereinbarung des Produkts bereitgestellt wird. Finden Sie unter der jeweiligen Produktdokumentation für Weitere Informationen.
