# <a name="visual-foxpro-terminology"></a>Visual FoxPro-Terminologie
**database**  
 In der Visual FoxPro, eine Datenbankdatei hat die Erweiterung DBC und kann eine oder mehrere enthalten **Tabellen**.  
  
 **Datenbanktabelle**  
 In der Visual FoxPro ist eine Tabelle, die einer Datenbank zugeordnet. Kontrast **freie Tabelle**.  
  
 **freie Tabelle**  
 In der Visual FoxPro, eine Tabelle, die nicht mit einer Datenbank verknüpft ist.  
  
 Erstellt eine DBF-Datei in FoxPro-Version 2.x ist eine kostenlose Tabelle, es sei denn, er wird in einer Visual FoxPro-Tabelle konvertiert und einer Visual FoxPro-Datenbank hinzugefügt. Kontrast **Datenbanktabelle**.  
  
 **SQL-Anweisung, die vorbereitbaren Objekts**  
 Eine SQL-Anweisung, die nicht von bereits verarbeitet wurde die **SQLPrepare** Funktion. Weitere Informationen zu dieser Funktion mithilfe der Visual FoxPro-ODBC-Treibers finden Sie unter [SQLPrepare (Visual FoxPro-ODBC-Treiber)](../../odbc/microsoft/sqlprepare-visual-foxpro-odbc-driver.md).  
  
 **table**  
 In der Visual FoxPro werden die Datensätze in einer Tabelle gespeichert. Jede Zeile einer Tabelle stellt einen Datensatz, und die Spalten der Tabelle dar, die Felder des Datensatzes. Jedes Visual FoxPro-Tabelle wird in einer eigenen Datei mit einer DBF-Erweiterung gespeichert. Visual FoxPro-Tabellen können mit einer Datenbank zugeordnet werden.  
  
 FoxPro-Version 2. *x* Tabellen sind nicht mit einer Datenbank verknüpft.
