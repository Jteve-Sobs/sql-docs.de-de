# <a name="read-only-status-text-file-driver"></a>Status „Schreibgeschützt“ (Textdateitreiber)
Wenn der Text-Treiber verwendet wird, sind die Datenquellensicht schreibgeschützt. Allerdings können Anwendungen für Text-Tabellen Ausführen von Einfügevorgängen.
