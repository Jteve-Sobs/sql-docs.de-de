# <a name="parameterized-query-limitations"></a>Einschränkungen für parametrisierte Abfragen
Wenn der Microsoft Access-Treiber verwendet wird, kann eine parametrisierte Abfrage mit der folgenden Syntax aufgerufen werden: *CALL-Abfragename* [(*Parameter*[,*Parameter*]...)].
