# <a name="installing-the-software-odbc"></a>Installieren der Software (ODBC)
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Version von Windows entfernt. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den von Oracle bereitgestellten ODBC-Treiber.  
  
 Der ODBC-Treiber für Oracle ist eine der Datenzugriffs Komponenten. Sie begleitet andere ODBC-Komponenten, z. b. den ODBC-Datenquellen-Administrator und sollte bereits installiert sein. Der Treiber finden Sie auch unter "Treiber und andere Downloads" auf der Microsoft-Produkt Support Services-Online Website unter [www.Microsoft.com](https://www.microsoft.com).  
  
 Netzwerk Software muss entsprechend der eigenen Dokumentation installiert werden. Der ODBC-Treiber für Oracle erfordert keine besonderen Überlegungen zur Installation, solange die Netzwerk Software unterstützt wird.  
  
 Oracle-Software muss entsprechend der eigenen Dokumentation installiert werden. Der ODBC-Treiber für Oracle erfordert im Allgemeinen keine besonderen Überlegungen zur Installation, solange der Treiber die Version unterstützt. Um die Kompatibilität von Produkten zu gewährleisten, installieren Sie den ODBC-Treiber für Oracle, um sicherzustellen, dass Sie über die neueste Version des Treibers verfügen. Oracle verwaltet eine öffentliche FTP-Site, auf der unter anderem Patches für die Oracle-Server Produkte und die Client Komponente, die im Lieferumfang der Server Produkte enthalten ist, veröffentlicht werden. Diese Patches sind für die ordnungsgemäße Funktionsweise mehrerer Microsoft-Produkte und-Technologien erforderlich. Weitere Informationen zu diesem Standort finden Sie unter [Oracle-Softwarepatches](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  Die Installation von Oracle-Software über MDAC/Windows DAC kann die aktuellen Versionen von MDAC überschreiben. Wenn bei der Verwendung von ODBC-Komponenten Probleme auftreten, installieren Sie MDAC neu.
