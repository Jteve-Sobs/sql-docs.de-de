# <a name="installing-the-software-odbc"></a>Installieren der Software (ODBC)
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Windows-Version entfernt. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den von Oracle bereitgestellten ODBC-Treiber.  
  
 Der ODBC-Treiber für Oracle ist eine der Datenzugriffskomponenten. Es begleitet andere ODBC-Komponenten, z. B. den ODBC-Datenquellenadministrator, und sollte bereits installiert sein. Der Treiber finden Sie auch unter "Treiber und andere Downloads" auf der Microsoft Product Support Services Online-Website unter [www.microsoft.com](https://www.microsoft.com).  
  
 Netzwerksoftware muss gemäß eigener Dokumentation installiert werden. Der ODBC-Treiber für Oracle erfordert keine besonderen Installationsüberlegungen, solange die Netzwerksoftware unterstützt wird.  
  
 Oracle-Software muss gemäß eigener Dokumentation installiert werden. Der ODBC-Treiber für Oracle erfordert in der Regel keine besonderen Installationsüberlegungen, solange der Treiber die Version unterstützt. Um jedoch produkte kompatibel zu halten, installieren Sie den ODBC-Treiber für Oracle zuletzt, um sicherzustellen, dass Sie über die neueste Version des Treibers verfügen. Oracle unterhält eine öffentliche FTP-Site, auf der unter anderem Patches an die Oracle-Serverprodukte und die Clientkomponente bereitgestellt werden, die mit den Serverprodukten ausgeliefert wird. Diese Patches sind für das ordnungsgemäße Funktionieren mehrerer Microsoft-Produkte und -Technologien erforderlich. Weitere Informationen zu dieser Website finden Sie unter [Oracle Software Patches](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  Die Installation von Oracle-Software über MDAC/Windows DAC kann aktuelle Versionen von MDAC überschreiben. Wenn Probleme mit ODBC-Komponenten auftreten, installieren Sie MDAC neu.
