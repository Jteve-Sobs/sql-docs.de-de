# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (Desktop-Datenbanktreiber)
Die Lesezeichen, die von einer *fOption* von SQL_GETBOOKMARK zurückgegeben werden, sind nur gültig, während die Abfrage geöffnet ist, und werden ungültig, wenn die Abfrage erneut ausgestellt wird. Persistente Lesezeichen werden nicht unterstützt.
