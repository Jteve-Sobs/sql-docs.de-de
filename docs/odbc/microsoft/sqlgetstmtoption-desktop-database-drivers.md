# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (Desktop-Datenbanktreiber)
Die Lesezeichen, die von einer *fOption* von SQL_GETBOOKMARK zurückgegeben werden, sind nur gültig, wenn die Abfrage geöffnet ist, und werden für ungültig erklärt, wenn die Abfrage neu ausgestellt wird. Persistente Lesezeichen werden nicht unterstützt.
