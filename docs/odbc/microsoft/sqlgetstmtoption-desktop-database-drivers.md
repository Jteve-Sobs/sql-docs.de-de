# <a name="sqlgetstmtoption-desktop-database-drivers"></a>SQLGetStmtOption (Desktop-Datenbanktreiber)
Die vom Lesezeichen ein *fOption* von SQL_GETBOOKMARK sind nur gültig, während die Abfrage geöffnet ist und für ungültig erklärt werden, wenn die Abfrage erneut ausgestellt. Persistente Lesezeichen werden nicht unterstützt.
