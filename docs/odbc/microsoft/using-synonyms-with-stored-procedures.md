# <a name="using-synonyms-with-stored-procedures"></a>Verwenden von Synonymen mit gespeicherten Prozeduren
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Version von Windows entfernt. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den von Oracle bereitgestellten ODBC-Treiber.  
  
 Der Microsoft ODBC-Treiber für Oracle-Versionen 2,0 und 2,5 unterstützt keine Synonyme beim Aufrufen von gespeicherten Oracle-Prozeduren. Synonyme funktionieren erwartungsgemäß, wenn Sie mit anderen Oracle-Datenbankobjekten wie Tabellen verwendet werden.
