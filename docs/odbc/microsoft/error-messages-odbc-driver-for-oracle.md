# <a name="error-messages-odbc-driver-for-oracle"></a>Fehlermeldungen (ODBC-Treiber für Oracle)
> [!IMPORTANT]  
>  Dieses Feature wird in einer zukünftigen Version von Windows entfernt werden. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den ODBC-Treiber, die von Oracle bereitgestellt.  
  
 Wenn ein Fehler auftritt, den Microsoft ODBC-Treiber für Oracle den SQLSTATE (ODBC-Fehlercode) und eine Fehlermeldung zurück. Der Treiber leitet diese Informationen sowohl von Fehlern, die vom Treiber erkannt wurden und von Fehlern, die von der Oracle-Server zurückgegeben.
