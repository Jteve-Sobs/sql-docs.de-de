# <a name="error-messages-odbc-driver-for-oracle"></a>Fehlermeldungen (ODBC-Treiber für Oracle)
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Version von Windows entfernt. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den von Oracle bereitgestellten ODBC-Treiber.  
  
 Wenn ein Fehler auftritt, gibt der Microsoft ODBC-Treiber für Oracle SQLSTATE (einen ODBC-Fehlercode) und eine Fehlermeldung zurück. Der Treiber leitet diese Informationen sowohl von Fehlern ab, die vom Treiber erkannt werden, als auch vom Oracle-Server zurückgegebene Fehler.
