# <a name="error-messages-odbc-driver-for-oracle"></a>Fehlermeldungen (ODBC-Treiber für Oracle)
> [!IMPORTANT]  
>  Diese Funktion wird in einer zukünftigen Windows-Version entfernt. Nutzen Sie diese Funktionen bei Neuentwicklungen nicht mehr, und planen Sie die Änderung von Anwendungen, die diese Funktion zurzeit verwenden. Verwenden Sie stattdessen den von Oracle bereitgestellten ODBC-Treiber.  
  
 Wenn ein Fehler auftritt, gibt der Microsoft ODBC-Treiber für Oracle den SQLSTATE (odBC-Fehlercode) und eine Fehlermeldung zurück. Der Treiber leitet diese Informationen sowohl von Fehlern ab, die vom Treiber erkannt wurden, als auch von Fehlern, die vom Oracle Server zurückgegeben wurden.
