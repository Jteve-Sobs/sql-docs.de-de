# <a name="alter-table-statement"></a>ALTER TABLE-Anweisung
Wenn die Microsoft Access, dBASE oder Paradoxdriver verwendet wird, wird die ALTER TABLE-Anweisung mit Add- und Drop Table Definition Einschränkungsklauseln unterstützt. (Für die Treiber für Microsoft Excel- oder Textdateien sind ALTER TABLE-Anweisungen nicht unterstützt.)  
  
> [!NOTE]  
>  Wenn Sie die Paradox-Treiber verwenden, ohne Sie zu der Datenbank-Engine für Borland implementieren, werden die ALTER TABLE-Anweisungen nicht unterstützt. Nur Lese- und anfügeereignisse-Anweisungen sind zulässig.
