# <a name="setting-up-the-visual-foxpro-odbc-driver"></a>Einrichten des Visual FoxPro-ODBC-Treibers
Informationen zum Herunterladen und Installieren des Visual FoxPro-ODBC-Treibers finden Sie unter [Visual FoxPro-ODBC-Treiber](https://go.microsoft.com/fwlink/?LinkId=121318) Downloads für MSDN.  
  
 Verwenden Sie das Microsoft Visual FoxPro-ODBC-Treiber-Setup-Programm, um die folgenden Schritte ausführen:  
  
-   Neue Komponenten hinzufügen.  
  
-   Entfernen Sie die installierte Komponenten.  
  
-   Installieren Sie zum Wiederherstellen von fehlenden Dateien und Einstellungen neu.  
  
-   Entfernen Sie alle zuvor installierte Komponenten.  
  
 Nach der Installation des Treibers auf Ihrem System wird das Setup-Programm erkennt die installierten Treiber-Komponenten und stellt zusätzliche Dialogfelder, die Sie zum Ändern des Treibers Konfiguration aktivieren.
