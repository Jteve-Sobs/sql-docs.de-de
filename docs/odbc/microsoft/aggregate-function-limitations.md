# <a name="aggregate-function-limitations"></a>Einschränkungen für Aggregatfunktionen
Eine Aggregatfunktion und ein nicht aggregierter Spaltenverweis können nicht beide als Argumente für eine einzelne SQL-Anweisung verwendet werden.
