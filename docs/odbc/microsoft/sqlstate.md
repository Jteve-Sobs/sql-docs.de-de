# <a name="sqlstate"></a>SQLSTATE
Bei Fehlern, die in der Datenquelle auftreten, ordnet der ODBC-Treiber den zurückgegebenen systemeigenen Fehler dem entsprechenden SQLSTATE zu. Für Fehler, die vom Treiber oder Treiber-Manager erkannt werden, generiert der ODBC-Treiber oder Treiber-Manager den entsprechenden SQLSTATE.
