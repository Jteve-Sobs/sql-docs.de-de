# <a name="date-arithmetic"></a>Datumsarithmetik
Der Treiber unterstützt das Hinzufügen und Subtrahieren einer ganzzahligen Datei aus einer DATE-, TIME- oder TIMESTAMP-Spalte. Für eine DATE-Spalte gibt die ganze Zahl die Anzahl der Tage an, die hinzugefügt oder subtrahiert werden sollen. Für eine TIME- oder TIMESTAMP-Spalte gibt die ganze Zahl die Anzahl der Sekunden an, die hinzugefügt oder subtrahiert werden sollen.
