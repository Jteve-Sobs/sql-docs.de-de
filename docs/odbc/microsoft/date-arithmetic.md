# <a name="date-arithmetic"></a>Datumsarithmetik
Der Treiber unterstützt das Hinzufügen und subtrahieren einer Ganzzahl von einer Datums-, Uhrzeit-oder timestamp-Spalte. Für eine Datums Spalte gibt die ganze Zahl die Anzahl der Tage an, die hinzugefügt oder subtrahiert werden soll. Für eine Zeit-oder timestamp-Spalte gibt die ganze Zahl die Anzahl der Sekunden an, die addiert oder subtrahiert werden soll.
