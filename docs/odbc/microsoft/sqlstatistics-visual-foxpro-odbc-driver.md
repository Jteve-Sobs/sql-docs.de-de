# <a name="sqlstatistics-visual-foxpro-odbc-driver"></a>SQLStatistics (Visual FoxPro-ODBC-Treiber)
> [!NOTE]  
>  Dieses Thema enthält Visual FoxPro-ODBC-Treiber-spezifische Informationen. Allgemeine Informationen zu dieser Funktion finden Sie unter den entsprechenden Themen unter [ODBC-API-Referenz](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Support: Vollständig  
  
 ODBC-API-Übereinstimmung: Ebene 1  
  
 Ruft eine Liste der Statistiken zu einer einzelnen Tabelle und die Indizes oder die Tag-Namen, die in der Tabelle zugeordnet. Der Treiber gibt zurück, die Informationen als Resultset.  
  
 Weitere Informationen finden Sie unter [SQLStatistics](../../odbc/reference/syntax/sqlstatistics-function.md) in die *ODBC Programmer's Reference*.
