# <a name="group-by-expression-list"></a>GROUP BY-Ausdrucksliste
GROUP BY unterstützt eine Ausdrucksliste sowie einen Spaltennamen. Die Auswahlliste einer SELECT-Anweisung mit einer GROUP BY-Klausel kann nur Ausdrücke aus der GROUP BY-Klausel oder Set-Funktionen enthalten.
