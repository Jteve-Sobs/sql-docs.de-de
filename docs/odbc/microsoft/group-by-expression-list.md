# <a name="group-by-expression-list"></a>GROUP BY-Ausdrucksliste
Group by unterstützt eine Ausdrucks Liste und einen Spaltennamen. Die SELECT-Liste einer SELECT-Anweisung mit einer Group By-Klausel kann nur Ausdrücke aus der Group By-Klausel oder Set-Funktionen enthalten.
