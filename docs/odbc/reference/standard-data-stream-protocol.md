# <a name="standard-data-stream-protocol"></a>Standardmäßiges Datenstromprotokoll
Ein Data Stream-Protokoll ist eine Möglichkeit, den Zugriff auf Daten in heterogenen DBMS. In der Tat enthält bereits ein Data Stream-Protokoll:

- Dem Standard ANSI/ISO Remote Datenbank Remotedatenzugriff (RDA): [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Obwohl die ANSI/ISO-System Promise anzeigt, ist es nicht weit heute implementiert.
