# <a name="standard-data-stream-protocol"></a>Standardmäßiges Datenstromprotokoll
Ein Standard-Datenstrom Protokoll ist eine Möglichkeit für den Zugriff auf Daten in heterogenen DBMSs. Tatsächlich ist ein Standard-Datenstrom Protokoll bereits vorhanden:

- Der ANSI/ISO-RDA-Standard (Remote Database Access): [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Obwohl das ANSI/ISO-System eine Zusage anzeigt, ist es noch heute nicht weit verbreitet.
