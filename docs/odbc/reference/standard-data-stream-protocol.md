# <a name="standard-data-stream-protocol"></a>Standardmäßiges Datenstromprotokoll
Ein Standarddatenstromprotokoll ist eine Möglichkeit, auf Daten in heterogenen DBMS zuzugreifen. Tatsächlich ist bereits ein Standarddatenstromprotokoll vorhanden:

- Der AnSI/ISO Remote Database Access (RDA)-Standard: [ISO/IEC 9579:2000](https://www.iso.org/iso/catalogue_detail.htm?csnumber=30615). Obwohl das ANSI/ISO-System vielversprechend ist, ist es heute nicht weit verbreitet.
