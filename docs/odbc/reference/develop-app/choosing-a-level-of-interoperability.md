# <a name="choosing-a-level-of-interoperability"></a>Auswählen einer Interoperabilitätsebene
Angenommen, die Anwendung verwendet ODBC. der nächste Schritt besteht darin, zu bestimmen, welche Interoperabilitäts Ebene erforderlich ist. Das grundlegende Maß an Interoperabilität ist normalerweise eine Funktion des Anwendungs Typs: benutzerdefinierte Anwendungen sind tendenziell nicht interoperabel, vertikale Anwendungen sind tendenziell in einer begrenzten Anzahl von DBMSs interoperabel, und generische Anwendungen sind tendenziell interoperabel. unter allen DBMSs.
