# <a name="diagnostic-tools"></a>Diagnosetools
Zwei Einrichtungen können bei der Diagnose von Problemen in ODBC-Anwendungen helfen. Beide sind vom ODBC-Administrator verfügbar und werden vom Treiber-Manager implementiert. *Die Ablaufverfolgung* bietet eine Möglichkeit, die Sequenz der Funktionsaufrufe in einer Protokolldatei aufzuzeichnen. Visual Studio Analyzer ermöglicht die Analyse von Informationen über die Interaktion von Komponenten in einer verteilten Umgebung.  
  
 In diesem Abschnitt werden die folgenden Themen behandelt:  
  
-   [Ablaufverfolgung](../../../odbc/reference/develop-app/tracing.md)  
  
-   [Visual Studio Analyzer](../../../odbc/reference/develop-app/visual-studio-analyzer.md)
