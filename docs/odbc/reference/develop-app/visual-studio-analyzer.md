# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Die Unterstützung für Visual Studio Analyzer wurde ab Windows 8 entfernt (Visual Studio Analyzer war nur in früheren Versionen von Visual Studio enthalten.) Verwenden Sie für einen alternativen Mechanismus zur Problembehandlung die Auftrags Ablauf Verfolgung.  
  
 Microsoft® Visual Studio™ Analyzer bietet eine allgemeine Übersicht über die ODBC-Anwendung. Der Entwickler kann die Struktur, Leistung und Interaktionen aus der Sicht der Anwendung und nicht aus einer Komponenten-oder Code Perspektive auswerten, analysieren und Debuggen. Die von Visual Studio Analyzer gesammelten Informationen sind in Form von Ereignissen, die eine Art Interaktion zwischen zwei Komponenten der Anwendung darstellen. Ereignisse können aus Funktionsaufrufen bestehen und von Funktionen zurückgegeben werden, wie z. b. Verbindungen, Datenbankabfragen und Transaktionen.
