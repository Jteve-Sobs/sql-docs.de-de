# <a name="visual-studio-analyzer"></a>Visual Studio Analyzer
> [!IMPORTANT]  
>  Die Unterstützung für Visual Studio Analyzer wurde ab Windows 8 entfernt (Visual Studio Analyzer war nur in älteren Versionen von Visual Studio enthalten). Verwenden Sie für einen alternativen Fehlerbehebungsmechanismus die BID-Ablaufverfolgung.  
  
 Microsoft® Visual Studio™ Analyzer bietet eine Ansicht auf hoher Ebene der ODBC-Anwendung. Der Entwickler kann die Struktur, Leistung und Interaktionen aus der Perspektive der Anwendung auswerten, analysieren und debuggen, anstatt aus einer Komponenten- oder Codeperspektive. Die von Visual Studio Analyzer gesammelten Informationen sind in Form von Ereignissen, die eine Art Interaktion zwischen zwei Komponenten der Anwendung darstellen. Ereignisse können aus Funktionsaufrufen und -rückgaben von Funktionen wie Verbindungen, Datenbankabfragen und Transaktionen bestehen.
