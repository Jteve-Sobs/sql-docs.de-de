# <a name="odbc-sql-grammar"></a>ODBC-SQL-Grammatik
Interoperable Anwendungen ausführen können, sollten immer die ODBC-SQL-Grammatik in SQL-Anweisungen verwenden. Beträchtliche Abweichung ist jedoch möglich, selbst innerhalb dieser Grammatik. Weitere Informationen finden Sie unter [Interoperabilität von SQL-Anweisungen](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
