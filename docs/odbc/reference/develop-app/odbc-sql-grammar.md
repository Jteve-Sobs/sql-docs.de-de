# <a name="odbc-sql-grammar"></a>ODBC-SQL-Grammatik
Interoperable Anwendungen sollten immer die ODBC SQL-Grammatik in SQL-Anweisungen verwenden. Auch innerhalb dieser Grammatik sind erhebliche Abweichungen möglich. Weitere Informationen finden Sie unter [Interoperabilität von SQL-Anweisungen](../../../odbc/reference/develop-app/interoperability-of-sql-statements.md).
