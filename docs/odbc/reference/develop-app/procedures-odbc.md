# <a name="procedures-odbc"></a>Prozeduren (ODBC)
Eine *Prozedur* ist ein ausführbares Objekt, das in der Datenquelle gespeichert ist. In der Regel handelt es sich dabei um eine oder mehrere vorkompilierte SQL-Anweisungen.
