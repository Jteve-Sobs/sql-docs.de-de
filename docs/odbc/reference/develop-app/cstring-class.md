# <a name="cstring-class"></a>CString-Klasse
Da Objekte der **CString** -Klasse in Microsoft® Visual C++® signiert sind und Zeichen folgen Argumente in ODBC-Funktionen nicht signiert sind, werden Anwendungen, die **CString** -Objekte ohne Umwandlung an ODBC-Funktionen übergeben, von Compilerwarnungen empfangen.
