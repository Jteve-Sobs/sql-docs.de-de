# <a name="data-types-in-odbc"></a>Datentypen in ODBC
ODBC verwendet zwei Sätze von Datentypen: SQL-Datentypen und C-Datentypen. SQL-Datentypen werden in der Datenquelle verwendet, und C-Datentypen werden in der C-Code in der Anwendung verwendet.
