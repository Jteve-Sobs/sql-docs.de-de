# <a name="tables-and-views"></a>Tabellen und Sichten
In ODBC-Funktionen sind Tabellen und Ansichten austauschbar. Die *Begriffstabelle* wird sowohl für Tabellen als auch für Ansichten verwendet, es sei denn, die *Begriffsansicht* wird explizit verwendet.
