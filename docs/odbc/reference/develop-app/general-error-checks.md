# <a name="general-error-checks"></a>Überprüfungen auf allgemeine Fehler
Der Treiber-Manager überprüft einen allgemeinen Fehler. Es gibt immer SQL_ERROR zurück, wenn der folgende Fehler auftritt: Die Funktion muss vom Treiber unterstützt werden.
