# <a name="numeric-literals"></a>Numerische Literale
Numerische Literale werden verwendet, wenn numerische Datenwerte in Zeichenfolgen gespeichert werden. Für die Konvertierung numerischer SQL-Daten in eine SQL_C_CHAR Zeichenfolge oder die Konvertierung numerischer C-Daten in eine SQL_CHAR- oder SQL_VARCHAR-Zeichenfolge wird die numerische Literalsyntax verwendet, um anzugeben, was im Ziel gespeichert ist. Für die Konvertierung einer numerischen, als SQL_C_CHAR Zeichenfolge in numerische SQL-Daten oder als SQL_CHAR-Zeichenfolge gespeicherten numerischen in numerische C-Daten wird diese Syntax verwendet, um zu überprüfen, was in der Quelle gespeichert ist.  
  
 Numerische Literale sollten der Syntax entsprechen, die im Abschnitt [Numerische Literale Syntax](../../../odbc/reference/appendixes/numeric-literal-syntax.md) in Anhang C: SQL-Grammatik definiert ist.
