# <a name="sql-server-help-and-feedback"></a>SQL Server-Hilfe und -Feedback
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md.md](../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

Wir sind an Ihren Ideen zur Verbesserung unserer Produkte und Dokumentation interessiert. Wir können nicht immer persönlich Antworten, aber wir stellen sicher, dass Ihr Feedback an die richtige Person bzw. das zuständige Team weitergeleitet wird. 

## <a name="i-want-to-give-feedback-on-sql-server"></a>Ich möchte Feedback zu SQL Server abgeben
Das SQL Server-Technikteam hält ein Auge auf dem SQL Server[Benutzerfeedback](https://feedback.azure.com/forums/908035-sql-server). Wenn Sie ein Problem mit SQL Server melden möchten, z. B. einen Absturz, Leistungseinbußen, unerwartetes Verhalten usw., ist das Benutzerfeedback der richtige Ort hierfür. 

Das [Benutzerfeedback](https://feedback.azure.com/forums/908035-sql-server) ist außerdem ein hervorragender Ort, um dem Technikteam mitzuteilen, was es besser machen kann. Wenn Sie Empfehlungen oder Vorschläge für Funktionen haben, teilen Sie diese ebenfalls hier mit. 

## <a name="i-want-to-make-a-suggestion-about-sql-server-management-studio"></a>Ich möchte einen Vorschlag zu SQL Server Management Studio machen
Das SQL Server Management Studio (SSMS)-Team überwacht ebenfalls eng das SQL Server-[Benutzerfeedback](https://feedback.azure.com/forums/908035-sql-server). Wenn Sie also Probleme zu melden oder Vorschläge zu unterbreiten haben, ist das Benutzerfeedback der richtig Ort hierfür. 

## <a name="i-want-to-contact-microsoft-support"></a>Ich möchte den Microsoft-Support kontaktieren
Wenn Sie ein Geschäftskunde sind, sehen Sie in den [Supportoptionen](https://support.microsoft.com/gp/support-options-for-business?forceorigin=esmc) nach.

Wenn Sie ein kommerzieller Benutzer sind, sehen Sie auf der Microsoft-Seite [Kontakt](https://support.microsoft.com/gp/contactus81?forceorigin=esmc&Audience=Commercial) nach.

## <a name="i-want-to-review-license-information-about-sql-server"></a>Ich möchte Lizenzinformationen zu SQL Server prüfen
Wenn Sie die Lizenzinformationen für SQL Server herunterladen und prüfen möchten, finden Sie diese unter [Zusätzliche Software-Lizenzbedingungen für Microsoft SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=39299).

## <a name="i-want-to-provide-feedback-for-sql-docs-content"></a>Ich möchte Feedback zu Inhalten der SQL-Dokumentation abgeben

### <a name="edit-contentincludesmediaedit-topic-pencilpng-editing-content"></a>![Inhalt bearbeiten](../includes/media/edit-topic-pencil.png) Bearbeiten von Inhalt
Haben Sie Fehler in unserer Dokumentation gefunden? Wussten Sie schon, dass Sie den Inhalt selbst bearbeiten könnten? Wenn Sie dies tun, wird unsere Dokumentation nicht nur besser, sondern Sie werden auch als Mitwirkender an der Seite aufgeführt. Weitere Informationen finden Sie unter [http://aka.ms/editsqldocs](http://aka.ms/editsqldocs). 

### <a name="surveys"></a>Umfragen
Wenn Sie etwas Zeit übrig haben und ein paar Gedanken zu unserer SQL-Dokumentation loswerden möchten, nehmen Sie an unserer Umfrage teil. Sie finden Sie unter [http://aka.ms/SQLDocSurvey](http://aka.ms/SQLDocSurvey), und sie sollte höchstens 10 bis 15 Minuten dauern. 

Wenn Sie dem SQL-Inhaltsteam etwas mitteilen möchten – eine Idee, einen Vorschlag, Feedback irgendetwas. Dies können Sie unter [http://aka.ms/SQLDocsFeedback](http://aka.ms/SQLDocsFeedback) machen.
