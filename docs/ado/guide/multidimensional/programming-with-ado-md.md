# <a name="programming-with-ado-md"></a>Programmieren mit ADO MD
Um ADO MD mit Ihrem Entwicklungs-Tool verwenden zu können, sollten Sie einen Verweis auf die Bibliothek mit ADO MD einrichten. Die Beschreibung des ADO MD-Bibliothek ist (Multi-dimensional) Microsoft ActiveX Data Objects-Bibliothek. Der Name der ADO MD-Bibliothek ist msadomd.dll, und die Programm-ID (ProgID) ist "ADOMD". Weitere Informationen dazu, wie Verweise auf Bibliotheken herstellen finden Sie in der Dokumentation zu Ihrem Entwicklungstool.
