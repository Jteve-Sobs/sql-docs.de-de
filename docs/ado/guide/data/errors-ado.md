# <a name="errors-ado"></a>Fehler (ADO)
Jeder Vorgang, der ADO-Objekte umfasst, kann einen oder mehrere Anbieter Fehler generieren. Wenn jeder Fehler auftritt, werden mindestens ein **Fehler** Objekt in die **Fehler** Auflistung des **Verbindungs** Objekts eingefügt. Ausführliche Informationen zur Behandlung von Warnungen und Fehlern in Ihrer ADO-Anwendung finden Sie unter [Fehlerbehandlung](../../../ado/guide/data/error-handling.md).  
  
 Anwendungsfehler können durch einen separaten Mechanismus ausgelöst werden. Beispielsweise enthält das **Err** -Objekt in Visual Basic Fehler auf Anwendungsebene.
