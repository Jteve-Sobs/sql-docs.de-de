# <a name="referencing-the-ado-libraries"></a>Verweisen auf die ADO-Bibliotheken
Die neueste Version von ADO ist als *"MSADO15. dll*verpackt. Die neuesten Versionen von ADO MD und ADOX werden als " *msadom. dll* " bzw. " *Msadox. dll*" verpackt. Diese Bibliotheken werden standardmäßig in *$installDir*installiert, wobei *$installDir* für den Pfad des Verzeichnisses steht, in dem die ADO-Bibliothek auf dem Computer installiert wurde. Um die ADO-Bibliotheken in Ihrer Anwendung zu verwenden, müssen Sie explizit im Anwendungsprojekt darauf verweisen.  
  
 Es folgen die Schritte, die Sie ausführen können, um auf die ADO-Bibliotheken zu verweisen:  
  
-   [In einer Visual Basic Anwendung](../../ado/guide/referencing-the-ado-libraries-in-a-visual-basic-6-application.md)  
  
-   [In einer Visual C++ Anwendung](../../ado/guide/referencing-the-ado-libraries-in-a-visual-c-application.md)
