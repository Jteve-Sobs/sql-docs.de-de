# <a name="membertypeenum"></a>MemberTypeEnum
Gibt die Einstellung für die [Type](../../../ado/reference/ado-md-api/type-property-ado-md.md) -Eigenschaft eines [Member](../../../ado/reference/ado-md-api/member-object-ado-md.md) -Objekts an.  
  
|Dauerhaft|value|BESCHREIBUNG|  
|--------------|-----------|-----------------|  
|**admembership**|4|Gibt an, dass das **Member** -Objekt alle Elemente der Ebene darstellt.|  
|**admembership-Formel**|3|Gibt an, dass das **Member** -Objekt mit einem Formel Ausdruck berechnet wird.|  
|**admembership-Measure**|2|Gibt an, dass das **Member** -Objekt zur Measures-Dimension gehört und ein quantitatives Attribut darstellt.|  
|**admitgliedregular**|1|Default. Gibt an, dass das **Member** -Objekt eine Instanz einer Geschäfts Entität darstellt.|  
|**admitgliedunknown**|0|Der Typ des Members kann nicht bestimmt werden.|
