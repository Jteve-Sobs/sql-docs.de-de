# <a name="source-property-ado-md"></a>Source-Eigenschaft (ADO MD)
Gibt die Quelle für die Daten im [Cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md)an.  
  
## <a name="settings-and-return-values"></a>Einstellungen und Rückgabewerte  
 Legt eine **Variante**fest oder gibt Sie zurück, und ist für geschlossene [Cellset](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) -Objekte Lese-/Schreibzugriff und für geöffnete **Cellset** -Objekte schreibgeschützt. Die **Variante** sollte eine gültige **Zeichenfolge**, z. b. eine MDX-Abfrage, enthalten.
