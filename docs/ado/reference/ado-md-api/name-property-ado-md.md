# <a name="name-property-ado-md"></a>Name-Eigenschaft (ADO MD)
Gibt den Namen eines Objekts.  
  
## <a name="return-values"></a>Rückgabewerte  
 Gibt eine **Zeichenfolge** und ist schreibgeschützt.  
  
## <a name="remarks"></a>Hinweise  
 Sie können abrufen, die **Namen** Eigenschaft eines Objekts durch einen Ordnungszahlverweis, nach dem sehen Sie sich das Objekt direkt anhand Ihres Namens. Z. B. wenn `cdf.CubeDefs(0).Name` "Bobs Video Store", führt Sie in diesem Artikel [CubeDef](../../../ado/reference/ado-md-api/cubedef-object-ado-md.md) als `cdf.CubeDefs("Bobs Video Store")`.
