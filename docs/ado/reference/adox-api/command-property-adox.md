# <a name="command-property-adox"></a>Command-Eigenschaft (ADOX)
Gibt an, ein ADO [Befehl](../../../ado/reference/ado-api/command-object-ado.md) -Objekt, das zum Erstellen oder Ausführen der Prozedur verwendet werden kann.  
  
## <a name="settings-and-return-values"></a>Einstellungen und Rückgabewerte  
 Legt fest oder gibt Sie ein gültigen ADO **Befehl** Objekt.  
  
## <a name="remarks"></a>Hinweise  
 Beim Abrufen und Festlegen dieser Eigenschaft, wenn der Anbieter keine persistenten Befehle unterstützt, wird ein Fehler auftreten.
