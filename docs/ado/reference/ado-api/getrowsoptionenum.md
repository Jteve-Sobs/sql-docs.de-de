# <a name="getrowsoptionenum"></a>GetRowsOptionEnum
Gibt an, wie viele Datensätze aus einem [Recordset](../../../ado/reference/ado-api/recordset-object-ado.md)abgerufen werden sollen.  
  
|Konstante|Wert|Beschreibung|  
|--------------|-----------|-----------------|  
|**adgetrowsrest**|-1|Ruft die restlichen Datensätze im **Recordset**entweder von der aktuellen Position oder einem Lesezeichen ab, das durch den *Start* -Parameter der [GetRows](../../../ado/reference/ado-api/getrows-method-ado.md) -Methode angegeben wird.|
