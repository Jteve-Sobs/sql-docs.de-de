# <a name="lineseparatorsenum"></a>LineSeparatorsEnum
Gibt das Zeichen an, das als Zeilen Trennzeichen in [textstreamobjekten](../../../ado/reference/ado-api/stream-object-ado.md) verwendet wird.  
  
|Dauerhaft|value|BESCHREIBUNG|  
|--------------|-----------|-----------------|  
|**adcr**|13|Gibt den Wagen Rücklauf an.|  
|**adCRLF**|-1|Default. Gibt Wagen Rücklauf-Zeilenvorschub an.|  
|**adlf**|10|Gibt den Zeilenvorschub an.|  
  
## <a name="adowfc-equivalent"></a>ADO/WFC-Entsprechung  
 Diese Konstanten haben keine ADO/WFC-Entsprechungen.
