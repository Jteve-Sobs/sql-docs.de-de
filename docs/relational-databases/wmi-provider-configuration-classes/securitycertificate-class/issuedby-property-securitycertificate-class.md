# <a name="issuedby-property-securitycertificate-class"></a>IssuedBy-Eigenschaft (SecurityCertificate-Klasse)

## <a name="syntax"></a>Syntax  
  
```  
  
object.IssuedBy [= value]  
```  
  
## <a name="parts"></a>Teile
