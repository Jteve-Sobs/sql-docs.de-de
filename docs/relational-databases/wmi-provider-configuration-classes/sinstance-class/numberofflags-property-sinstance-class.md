# <a name="numberofflags-property-sinstance-class"></a>NumberOfFlags-Eigenschaft (SInstance-Klasse)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Ruft die Anzahl der Flags für die-Instanz [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]ab.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object.NumberOfFlags [= value]  
```  
  
## <a name="parts"></a>Bestandteile  
 *Objekt*  
 Ein [SInstance-Klassenobjekt](../../../relational-databases/wmi-provider-configuration-classes/sinstance-class/sinstance-class.md) , das eine Serverinstanz darstellt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Ein **UInt32** -Wert, der die Anzahl von Flags für die- [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]Instanz angibt.
