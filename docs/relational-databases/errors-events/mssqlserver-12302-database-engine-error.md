# <a name="mssqlserver_12302"></a>MSSQLSERVER_12302
 [!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]
  
## <a name="details"></a>Details  
  
| attribute | Wert |  
| :-------- | :---- |  
|Produktname|SQL Server|  
|Ereignis-ID|12302|  
|Ereignisquelle|MSSQLSERVER|  
|Komponente|SQLEngine|  
|Symbolischer Name|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|Meldungstext|Das Aktualisieren von Spalten, die Teil der PRIMARY KEY-Einschränkung sind, wird nicht bei *construct* unterstützt.|  
  
## <a name="user-action"></a>Benutzeraktion  
Aktualisieren Sie keine Spalten, die Teil der PRIMARY KEY-Einschränkung sind.
