# <a name="database-maintenance-plan-stored-procedures-transact-sql"></a>Gespeicherte Prozeduren für Datenbank-Wartungspläne (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  [!INCLUDE[msCoName](../../includes/msconame-md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] unterstützt die folgenden gespeicherten System Prozeduren, die zum Einrichten von Wartungs Tasks verwendet werden. Diese gespeicherten Prozeduren werden mit Datenbank-Wartungsplänen verwendet. Diese Funktion wurde durch Wartungspläne ersetzt, die nicht diese gespeicherten Prozeduren verwenden. Verwenden Sie diese Verfahren, um Daten Bank Wartungspläne für Installationen beizubehalten, für die ein Upgrade [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]von einer früheren Version von durchgeführt wurde.
