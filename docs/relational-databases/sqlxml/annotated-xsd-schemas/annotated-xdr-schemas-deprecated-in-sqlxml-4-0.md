# <a name="annotated-xdr-schemas-deprecated-in-sqlxml-40"></a>XDR-Schemas mit Anmerkungen (in SQLXML 4.0 veraltet)
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
  XDR-Schemas mit Anmerkungen sind veraltet. Für diese Funktion wird künftig keine Entwicklungsarbeit mehr geleistet. Vermeiden Sie daher das Verwenden dieser Funktion bei neuen Entwicklungsarbeiten. Verwenden Sie stattdessen XSD-Schemas mit Anmerkungen. Sie können vorhandene XDR-Schemas mit Anmerkungen in XSD-Schemas konvertieren. Weitere Informationen finden Sie unter [konvertieren mit Anmerkungen versehen XDR-Schemas in entsprechende XSD-Schemas &#40;SQLXML 4.0&#41;](../../../relational-databases/sqlxml/annotated-xsd-schemas/converting-annotated-xdr-schemas-to-equivalent-xsd-schemas-sqlxml-4-0.md). Weitere Informationen zu den veralteten XDR-Schemas, finden Sie unter den [Dokumentation zu SQL Server 2008](https://go.microsoft.com/fwlink/?LinkId=202011).
