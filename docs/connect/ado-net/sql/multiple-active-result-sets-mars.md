# <a name="multiple-active-result-sets-mars"></a>Multiple Active Result Sets (MARS)

![Download-DownArrow-Circled](../../../ssdt/media/download.png)[ADO.NET herunterladen](../../sql-connection-libraries.md#anchor-20-drivers-relational-access)

Mars (Multiple Active Result Sets) ist ein Feature, das die Ausführung mehrerer Batches über eine einzelne Verbindung ermöglicht. In früheren Versionen konnte jeweils nur ein Batch für eine einzelne Verbindung ausgeführt werden. Das Ausführen mehrerer Batches mit Mars impliziert nicht die gleichzeitige Ausführung von Vorgängen.
