# <a name="sql-server-data-operations-in-adonet"></a>SQL Server-Datenvorgänge in ADO.NET

![Download-DownArrow-Circled](../../../ssdt/media/download.png)[ADO.NET herunterladen](../../sql-connection-libraries.md#anchor-20-drivers-relational-access)

In diesem Abschnitt werden SQL Server Features und Funktionen beschrieben, die für die Microsoft SqlClient-Datenanbieter für SQL Server (<xref:Microsoft.Data.SqlClient>) spezifisch sind.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
[Massenkopiervorgänge in SQL Server](bulk-copy-operations-sql-server.md)  
Beschreibt die Massenkopierfunktionen für den .NET-Datenanbieter für SQL Server.  
  
[Mehrere aktive Resultsets (MARS)](multiple-active-result-sets-mars.md)  
Beschreibt, wie mehrere <xref:Microsoft.Data.SqlClient.SqlDataReader> in einer Verbindung geöffnet werden, wenn jede Instanz von <xref:Microsoft.Data.SqlClient.SqlDataReader> von einem separaten Befehl aus gestartet wird.  
  
[Asynchrone Vorgänge](asynchronous-operations.md)  
Beschreibt das Ausführen asynchroner Datenbankoperationen mithilfe einer API, die auf der Grundlage des von .NET Framework verwendeten asynchronen Modells entwickelt wurde.  
  
[Tabellenwertparameter](table-valued-parameters.md)  
Beschreibt, wie mit Tabellenwert Parametern gearbeitet wird, die in SQL Server 2008 eingeführt wurden.
