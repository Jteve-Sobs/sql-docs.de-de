# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  Die IBCPSession2-Schnittstelle ist eine Erweiterung von IBCPSession, die eine Element Funktion bereitstellt, die eine Alternative zum Aufrufen von IBCPSession:: BCPColFmt für jede Spalte darstellt.  IBCPSession2 erbt von IBCPSession und fügt eine neue Methode hinzu: [IBCPSession2:: bcpsetbulkmode](../../oledb/ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
