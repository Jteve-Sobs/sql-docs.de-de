# <a name="icolumnsrowset"></a>IColumnsRowset
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]

[!INCLUDE[Driver_OLEDB_Download](../../../includes/driver_oledb_download.md)]

  Der OLE DB-Treiber für SQL Server fügt IColumnsRowset::GetColumnRowset die Spalte DBCOLUMN_BASETABLEINSTANCE hinzu. Diese Spalte gibt DBTYPE_I2 zurück, und sie ist für die Verwendung durch Microsoft reserviert. Die Informationen in dieser Spalte können in zukünftigen Versionen geändert werden.
