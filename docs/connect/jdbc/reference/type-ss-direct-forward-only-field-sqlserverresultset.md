# <a name="type_ss_direct_forward_only-field-sqlserverresultset"></a>TYPE_SS_DIRECT_FORWARD_ONLY-Feld (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Dient zum Angeben eines schnellen schreibgeschützten Vorwärtscursortyps für [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public static final int TYPE_SS_DIRECT_FORWARD_ONLY  
```  
  
## <a name="field-value"></a>Feldwert  
 Ein **int**-Wert von 2003.
