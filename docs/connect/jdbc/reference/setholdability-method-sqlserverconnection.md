## <a name="syntax"></a>Syntax  
  
```  
  
public void setHoldability(int nNewHold)  
```  
  
#### <a name="parameters"></a>Parameter  
 *nNewHold*  
  
 Ein Wert vom Typ **int** mit einer der folgenden Haltbarkeitsstufen:
