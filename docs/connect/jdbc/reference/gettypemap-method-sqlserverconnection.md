# <a name="gettypemap-method-sqlserverconnection"></a>getTypeMap-Methode (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Ruft das Map-Objekt ab, das diesem [SQLServerConnection](../../../connect/jdbc/reference/sqlserverconnection-class.md)-Objekt zugeordnet ist.  
  
> [!NOTE]  
>  Diese Methode wird von [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)] derzeit nicht unterstützt.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public java.util.Map getTypeMap()  
```  
  
## <a name="return-value"></a>Rückgabewert  
 Ein Map-Objekt  
  
## <a name="exceptions"></a>Ausnahmen  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Bemerkungen  
 Diese getTypeMap-Methode wird von der getTypeMap-Methode in der java.sql.Connection-Schnittstelle angegeben.
