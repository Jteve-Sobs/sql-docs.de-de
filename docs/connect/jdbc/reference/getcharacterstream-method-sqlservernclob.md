# <a name="getcharacterstream-method-sqlservernclob"></a>getCharacterStream-Methode (SQLServerNClob)

  Ruft die **NCLOB** Wert festgelegt, die von diesem **java.sql.NClob** als Objekt ein **java.io.Reader** Objekt.
