# <a name="sqlserverexception-constructor-javalangstring-javalangthrowable"></a>SQLServerException-Konstruktor (Java. lang. String, Java. lang. Throwable)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Initialisiert eine neue Instanz der [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md) -Klasse, wenn ein **Zeichen** folgen Objekt und ein **drobbares** Objekt angegeben werden.

## <a name="syntax"></a>Syntax  
  
```  
public SQLServerException(java.lang.String errText,
            java.lang.Throwable cause)
            
```  
  
#### <a name="parameters"></a>Parameter  
 *errText*  
  
 Eine Zeichenfolge, die den Fehlertext enthält.
 
 *cause*  
  
 Ein einschränktes Objekt, das die Ursache der Ausnahme enthält.
