# <a name="concur_ss_optimistic_ccval-field-sqlserverresultset"></a>CONCUR_SS_OPTIMISTIC_CCVAL-Feld (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Dient zum Angeben eines Typs der optimistischen Nebenläufigkeit (Lese-/Schreibzugriff) für [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] ohne Zeilensperre.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```  
  
## <a name="field-value"></a>Feldwert  
 Ein **int**-Wert von „1010“
