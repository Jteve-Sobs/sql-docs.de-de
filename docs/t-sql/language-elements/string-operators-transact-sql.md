# <a name="string-operators-transact-sql"></a>Zeichenfolgenoperatoren (Transact-SQL)
[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

[!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] stellt die folgenden Zeichenfolgenoperatoren bereit: Operatoren der Zeichenfolgenverkettung können zwei oder mehr der folgenden Datentypen zu einem Ausdruck kombinieren: 
* Zeichen- oder Binärzeichenfolgen
* Spalten 
* Kombination von Zeichenfolgen und Spaltennamen 

Operatoren mit Platzhalterzeichenfolgen können einem oder mehreren Zeichen in einem Zeichenfolgenvergleich entsprechen. LIKE und PATINDEX sind Beispiele für zwei dieser Vorgänge.
