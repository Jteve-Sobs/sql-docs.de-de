# <a name="sql-server-migration-assistant-for-db2-db2tosql"></a>SQL Server Migration Assistant für DB2 (DB2ToSQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Migration Assistant (SSMA) für DB2 ist ein Tool zum Migrieren von DB2 [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] -Daten [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Banken zu [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2012, [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2014, 2016, 2017 unter [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Windows und Linux, 2019 unter Windows und Linux oder Azure SQL-Datenbank. SSMA für DB2 konvertiert DB2-Daten Bank [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Objekte in Datenbankobjekte, erstellt [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]diese Objekte in und migriert dann Daten von [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] DB2 zu oder Azure SQL-Datenbank.  
  
In dieser Dokumentation finden Sie Informationen zu SSMA für DB2 und eine Schritt-für-Schritt-Anleitung zum Migrieren von DB2-Datenbanken zu [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. In der folgenden Tabelle finden Sie Artikel, die Ihnen helfen, mehr zu erfahren:
