# <a name="installing-ssma-for-mysql-mysqltosql"></a>Installieren von SSMA für MySQL (MySqlToSql)
SQL Server Migration Assistant (SSMA) for MySQL besteht aus einer Clientanwendung, die Sie verwenden zum Durchführen einer Migration von MySQL auf [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] oder SQL Azure. Sie enthält auch ein Erweiterungspaket aus, die Migration von Daten und die Verwendung von MySQL-Systemfunktionen in Ihren migrierten Datenbanken unterstützt.  
  
Installieren Sie die Client-Anwendung auf dem Computer aus dem Sie die Migrationsschritte ausgeführt werden. Sie müssen die Erweiterung-Pack-Dateien auf dem Computer installieren, in der die migrierten Datenbanken gehostet werden.  Dieser Computer muss SQL Server ausgeführt werden.  
  
## <a name="upgrading-ssma-for-mysql"></a>Aktualisieren von SSMA für MySQL  
Wenn Sie auf eine neuere Version von SSMA für MySQL aktualisieren möchten, müssen Sie zuerst deinstallieren des Clients und Server-Erweiterung Pack und klicken Sie dann die neue Version installieren.
