# <a name="sql-server-migration-assistant-for-access-accesstosql"></a>SQL Server Migration Assistant für den Zugriff (Access Token)

[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Migration Assistant (SSMA) für den Zugriff ist ein Tool zum Migrieren [!INCLUDE[msCoName](../../includes/msconame_md.md)] von Datenbanken von Zugriffs Versionen [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 97 bis [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2010 zu [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2012, [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2014, 2016, 2017 unter [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Windows und Linux, 2019 unter Windows [!INCLUDE[msCoName](../../includes/msconame_md.md)] und Linux oder Azure SQL-Datenbank. SSMA for Access konvertiert den Zugriff auf Daten [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Bank Objekte auf-oder Azure SQL-Daten Bank [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Objekte, lädt diese Objekte in oder Azure SQL-Datenbank und migriert Daten aus dem Zugriff auf [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] oder Azure SQL-Datenbank.
  
In dieser Dokumentation finden Sie Informationen zu SSMA für den Zugriff und eine Schritt-für-Schritt-Anleitung zum [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Migrieren von Access-Datenbanken zu oder Azure SQL-Datenbank sowie Informationen zu Problemen, die nach der Migration auftreten können.
