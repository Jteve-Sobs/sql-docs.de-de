# <a name="sql-server-migration-assistant-for-oracle-oracletosql"></a>SQL Server Migration Assistant für Oracle (oracleto SQL)
[!INCLUDE[msCoName](../../includes/msconame_md.md)][!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Migration Assistant (SSMA) für Oracle ist ein Tool zum Migrieren von Oracle [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] -Daten [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Banken zu [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2012, [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2014, 2016, 2017 unter [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Windows und Linux, 2019 unter Windows und Linux und Azure SQL-Datenbank. SSMA für Oracle konvertiert Oracle-Daten Bank [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Objekte in Datenbankobjekte, erstellt [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]diese Objekte in und migriert dann Daten von [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]Oracle zu.  
  
In dieser Dokumentation werden Sie mit SSMA für Oracle eingeführt und erhalten Schritt-für-Schritt-Anleitungen zum Migrieren von Oracle-Datenbanken zu [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. In der folgenden Tabelle finden Sie Artikel, die Ihnen helfen, mehr zu erfahren:
