# <a name="view-project-properties"></a>Anzeigen der Projekteigenschaften
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Sie können dem Projekt zugeordnete Eigenschaften im Eigenschaftenfenster anzeigen.  
  
### <a name="to-view-project-properties"></a>So zeigen Sie Projekteigenschaften an  
  
1.  Wählen Sie im Projektmappen-Explorer ein Projekt aus.  
  
2.  Wenn das Eigenschaftenfenster nicht geöffnet ist, klicken Sie im Menü **Ansicht** auf **Eigenschaftenfenster**.  
  
3.  Die Eigenschaften des Projekts werden im Eigenschaftenfenster angezeigt.  
  
Projekteigenschaften wie das Erstellungsdatum sind schreibgeschützt und können über das Eigenschaftenfenster nicht geändert werden.
