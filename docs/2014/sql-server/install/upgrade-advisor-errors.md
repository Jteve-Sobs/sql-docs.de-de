# <a name="upgrade-advisor-errors"></a>Fehler des Upgrade Advisors
  Fehler des Upgrade Advisor werden möglicherweise im Berichts-Viewer angezeigt. Derartige Fehler hängen mit diesem Hilfethema zusammen.  
  
 Sie können im Berichts-Viewer aufgelistete Probleme ignorieren, die Links zu diesem Thema enthalten.
