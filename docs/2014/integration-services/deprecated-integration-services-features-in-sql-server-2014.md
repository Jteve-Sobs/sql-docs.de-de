# <a name="deprecated-integration-services-features-in-sql-server-2014"></a>Als veraltet markierte Funktionen in SQL Server 2014 Integration Services
  In diesem Thema werden die veralteten [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] Features beschrieben, die in der aktuellen Version von noch verfügbar sind [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] . Diese Funktionen werden voraussichtlich in einer zukünftigen Version von [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]entfernt. Als veraltet markierte Funktionen sollten in neuen Anwendungen nicht verwendet werden.  
  
 Es gibt keine veralteten [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] -Funktionen in [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)].
