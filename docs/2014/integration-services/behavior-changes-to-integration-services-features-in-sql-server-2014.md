# <a name="behavior-changes-to-integration-services-features-in-sql-server-2014"></a>Verhaltensänderungen von Integration Services-Features in SQL Server 2014
  In diesem Thema werden Änderungen im Verhalten von [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)]beschrieben. Verhaltensänderungen wirken sich darauf aus, wie Funktionen in der aktuellen Version [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] von im Vergleich zu früheren Versionen [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]von funktionieren oder interagieren.  
  
 Die Integration Services-Funktionen in SQL Server 2014 weisen keine Verhaltensänderungen auf.
