# <a name="task-3-verifying-the-data-in-master-data-manager"></a>Aufgabe 3: Überprüfen der Daten im Master Data Manager
  In dieser Aufgabe überprüfen Sie, ob die **Supplier-Entität** auf **MDS** mit **Master Data Manager Web Application**erstellt wird.

1.  Wenn **Master Data Manager** bereits geöffnet ist, klicken Sie oben auf SQL Server **2012 Master Data Services,** um zur Startseite zu navigieren. Navigieren Sie `http://localhost/MDS` andernfalls zu, um **Master Data Manager**zu starten.

2.  Wählen Sie **Lieferanten** für **Modell**aus, und klicken Sie auf **Explorer**.

     ![Master Data Manager – Explorer (Schaltfläche)](../../2014/tutorials/media/et-verifyingthedatainmasterdatamanager.jpg "Master Data Manager – Explorer (Schaltfläche)")

3.  Überprüfen Sie die in MDS gespeicherten Daten. Wenn die Daten nicht angezeigt werden, vergewissern Sie sich, dass Sie auf der Startseite **Lieferanten** für das **Modell** ausgewählt haben, bevor Sie **Explorer**starten. Sie können der Lieferantenliste hinzufügen oder aus ihr löschen, indem Sie die Schaltflächen **Mitglied hinzufügen** und **Mitglied löschen** auf der Symbolleiste verwenden.
