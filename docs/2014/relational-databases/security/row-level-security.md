# <a name="row-level-security"></a>Sicherheit auf Zeilenebene
  Mithilfe der Sicherheit auf Zeilenbene können Kunden den Zugriff auf die Zeilen in einer Datenbanktabelle auf Grundlage der Merkmale des Benutzers steuern, der eine Abfrage ausführt (z. B. Gruppenmitgliedschaft oder Ausführungskontext). Sicherheit auf Zeilenebene ist jetzt in [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] 2016 verfügbar. Die aktuelle Beschreibung dieser Funktion finden Sie in der aktuellen Dokumentation [Sicherheit auf Zeilenebene](https://msdn.microsoft.com/library/dn765131.aspx) .
