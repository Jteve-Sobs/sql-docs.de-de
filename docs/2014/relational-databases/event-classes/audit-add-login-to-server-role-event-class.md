# <a name="audit-add-login-to-server-role-event-class"></a>Audit Add Login to Server Role-Ereignisklasse
  Die **Audit Add Login to Server Role** -Ereignisklasse tritt auf, wenn ein Anmelde Name einer Server Rolle mit fester Größe hinzugefügt oder daraus entfernt wird. Diese Ereignisklasse wird für gespeicherte Prozeduren **sp_addsrvrolemember** und **sp_dropsrvrolemember** verwendet.
