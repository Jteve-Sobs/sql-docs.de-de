# <a name="deprecation-event-category"></a>Veraltung (Ereigniskategorie)
  Die **Veraltung** -Ereigniskategorie enthält allgemeine Veraltungsereignisse.

|[Deprecation Final Support (Ereignisklasse)](deprecation-final-support-event-class.md)|Gibt an, dass diese Funktion in der nächsten Hauptversion von [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]entfernt wird.|
