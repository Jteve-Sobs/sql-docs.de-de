# <a name="server-event-category"></a>Server-Ereigniskategorie
  Die **Server** -Ereigniskategorie enthält allgemeine Serverereignisse.  
  
## <a name="in-this-section"></a>In diesem Abschnitt  
  
|Thema|BESCHREIBUNG|  
|-----------|-----------------|  
|[Mount Tape (Ereignisklasse)](mount-tape-event-class.md)|Gibt an, dass eine Anforderung zum Bandeinlegen empfangen wurde.|  
|[Server Memory Change-Ereignisklasse](server-memory-change-event-class.md)|Gibt an, dass die Speicherauslastung durch [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] geändert wurde.|  
|[Trace File Close (Ereignisklasse)](trace-file-close-event-class.md)|Gibt an, dass eine Ablaufverfolgungsdatei beim Rollover für Ablaufverfolgungsdateien geschlossen wurde.|
