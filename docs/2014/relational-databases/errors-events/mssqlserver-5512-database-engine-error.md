# <a name="mssqlserver5512"></a>MSSQLSERVER_5512
    
## <a name="details"></a>Details  
  
|||  
|-|-|  
|Produktname|MSSQLSERVER|  
|Ereignis-ID|5512|  
|Ereignisquelle|MSSQLSERVER|  
|Komponente|SQLEngine|  
|Symbolischer Name|FS_TRUNCATEDIR_FAILED|  
|Meldungstext|Fehler 0x%x (%ls) beim Abschneiden des Verzeichnisses ''%.*ls''.|  
  
## <a name="explanation"></a>Erklärung  
 Der Abschneidevorgang ist fehlgeschlagen.

 Die Ursache des Fehlers finden Sie im entsprechenden Windows-Statuscode. Weitere Informationen über diesen Statuscode finden Sie unter [Ereignis- und Fehlermeldungen, die Microsoft-Support](https://support.microsoft.com/search?query=events%20and%20error%20messages).
