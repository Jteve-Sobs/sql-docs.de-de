# <a name="properties-property-servernetworkprotocolipaddress-class"></a>Properties-Eigenschaft (ServerNetworkProtocolIPAddress-Klasse)
  Ruft die Eigenschaften ab, die einer IP-Adresse zugeordnet sind.  
  
## <a name="syntax"></a>Syntax  
  
```  
  
object  
.Properties [= value]  
```  
  
## <a name="parts"></a>Bestandteile  
 *Objekt*  
 Ein [ServerNetworkProtocolIPAdress-Klassenobjekt](servernetworkprotocolipaddress-class.md) , das eine IP-Adresse für das Netzwerkprotokoll in einer Instanz von [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]darstellt.  
  
## <a name="property-valuereturn-value"></a>Eigenschaftswert/Rückgabewert  
 Ein Array von Objekten, die die von der IP-Adresse unterstützten Eigenschaften darstellen.
