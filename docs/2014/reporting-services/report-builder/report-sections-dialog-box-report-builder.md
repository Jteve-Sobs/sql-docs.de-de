# <a name="report-sections-dialog-box-report-builder"></a>Berichtsabschnitte (Dialogfeld) (Berichts-Generator)
  Wenn Sie einen Bericht öffnen, überprüft Berichts-Generator, ob der Bericht nur einen RDL-Abschnitt (Report Definition Language) ** \<>** -Element enthält. Das Öffnen eines Berichts mit mehreren ** \<Abschnitten>** Elemente wird nicht unterstützt.  
  
 Obwohl eine Berichtsdefinition mit mehreren Abschnitten programmgesteuert generiert werden kann, wird die Bearbeitung eines solchen Berichtes in einer Berichterstellungsumgebung nicht unterstützt.  
  
 Informationen zum Ermitteln des aktuellen Berichtsdefinitionsformats für einen Bericht, einen Berichtsserver, ein Projekt oder die Berichterstellungsumgebung finden Sie unter [Suchen der Berichtsdefinitions-Schemaversion (SSRS)](../reports/find-the-report-definition-schema-version-ssrs.md).  
  
 Klicken Sie auf **OK**, um den Vorgang fortzusetzen.
