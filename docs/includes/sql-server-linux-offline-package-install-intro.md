Wenn Ihr Linux-Computer keinen Zugriff auf die Onlinerepositorys hat, die in den vorherigen Abschnitten verwendet werden, können Sie die Paketdateien direkt herunterladen. Diese Pakete befinden sich im Microsoft-Repository [https://packages.microsoft.com](https://packages.microsoft.com).
