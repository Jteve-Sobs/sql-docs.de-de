Der Downloadspeicherort für [!INCLUDE[ssnoversion](ssnoversion-md.md)] ist abhängig von der Edition.

* **SQL Server Enterprise-, Standard- und Express-Editionen** werden für die Verwendung in einer Produktionsumgebung lizenziert. Wenden Sie sich für die Installation von Medien für die Enterprise- und Standard-Editionen an Ihren Softwareanbieter. Einkaufsinformationen und ein Verzeichnis mit Microsoft-Partner finden Sie auf der [Einkaufswebsite von Microsoft](https://www.microsoft.com/en-us/server-cloud/products/sql-server/overview.aspx). 
* **Kostenlose Editionen** sind unter [SQL Server-Downloads](https://www.microsoft.com/sql-server/sql-server-downloads).
