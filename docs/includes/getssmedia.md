Der Downloadspeicherort für [!INCLUDE[ssnoversion](ssnoversion-md.md)] ist abhängig von der Edition.

* **SQL Server Enterprise-, Standard- und Express-Editionen** werden für die Verwendung in einer Produktionsumgebung lizenziert. Für die Enterprise Edition und Standard Edition und wenden Sie sich an den Anbieter Ihrer Software für den Installationsmedien. Einkaufsinformationen und ein Verzeichnis mit Microsoft-Partner finden Sie auf der [Einkaufswebsite von Microsoft](https://www.microsoft.com/en-us/server-cloud/products/sql-server/overview.aspx). 
* **Kostenlose Editionen** finden Sie unter [SQL Server-downloads](https://www.microsoft.com/sql-server/sql-server-downloads).
